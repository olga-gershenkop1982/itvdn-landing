# _ОРГАНИЗАЦИЯ РАБОЧЕГО ПРОЦЕССА_
## _ИНСТРУКЦИИ_

### _Перед созданием проекта, вам необходимо:_
* Проверить, есть ли у вас `node`, `npm`, `npx`:

> node -v

> npm -v

> npx -v

_или_

> node --version

> npm --version

> npx --version

* Если они установлены,
  вы увидите версию в терминале,
  например, `v12.14.1`.

> `node:`
> ```
> $ node -v
> или
> $ node --version
> v12.14.1
> ```

> `npm:`
> ```
> $ npm -v
> или
> $ npm --version
> 6.13.4
> ```

> `npx:`
> ```
> $ npx -v
> или
> $ npx --version
> 6.13.4
> ```

* Если они не установлены,
  следуйте [приведенным здесь инструкциям](https://nodejs.org/en/).
  
### _Перед установкой плагинов, вам необходимо:_

* Удалить старую версию `gulp`,
  если ранее вы устанавливали его глобально:

>> npm rm --global gulp
>
>_или_
>
>> npm rm -g gulp

### _Чтобы установить плагины, вам необходимо:_

>> npm install --save-dev `<package>`
> * _устанавливаем пакет_
>
>_или_
>
>> npm install --save-dev `<package>` `<package>`
> * _устанавливаем несколько пакетов одновременно_

_Например,_

>> npm install --save-dev `browser-sync`
> 
> _или_
> 
>> npm install --save-dev `browser-sync` `gulp` `gulp-pug` `gulp-sass`

* _если мы `не пишем никакие цифры`,
  то он будет устанавливать`последние версии`;_

> `-g`
> * _"глобально";_
> * _т.е. `browser-sync` будет доступен нам с консоли в любом месте;_

> `"локально"`
> * _устанавливаем в `текущий проект`;_

> `--save-dev`
> * _то, что нужно `для разработки`;_

> `--save`
> * _то, что нужно `для продакшена`;_

### _После установки плагинов, вам необходимо:_

* Проверить, есть ли у вас `gulp`:

>> gulp -v
>
>_или_
>
>> gulp --version

> _Если не увидите версию, например:_
> ```
> CLI version: 2.3.0
> Local version: 4.0.2
> ```
> _перезапустите терминал._

### _Чтобы запустить чужой проект, вам необходимо:_

* Склонировать репозиторий:

> `git clone <path>`

_Например,_

> _HTTPS:_
>
>> git clone https://github.com/olga-gershenkop1982/itvdn-landing.git

_или_

> _SSH:_
>
>> git clone git@github.com:olga-gershenkop1982/itvdn-landing.git

* Запустить `npm install`:

>> npm install
> 
> _установит в проект все библиотеки, пакеты и зависимости, указанные в `package.json`_

### _Чтобы найти что-либо в интернете:_

_в строке браузера:_
>> git download
> * _найти адрес `загрузки системы Git на компьютер`_

_в строке браузера:_
>> git tutorial
> * _найти обучающие `материалы по работе в Git`_

_в строке браузера:_
>> md syntax
> * _найти обучающие материалы по работе с `синтаксисом md`_

_в строке браузера:_
>> gulp
> * _найти официальный сайт `gulp`_
